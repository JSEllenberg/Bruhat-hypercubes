# Bruhat-hypercubes
Supplementary information for "Bruhat intervals that are large hypercubes"

The two files here are:

Alpha Evolve large d-invariant finder.py
The program developed by AlphaEvolve that takes an integer n as input and produces as output a pair of permutations in S_n with large d-invariant.

Large d-invariant Finder Prompt.txt
The full prompt provided to AlphaEvolve for the large d-invariant finding experiment.
